# Official Codex Structure

The Codex is organized into six major documentation volumes.

1. Game Bible — what the player experiences.
2. Technical Bible — how the engine works.
3. AI Bible — how people think.
4. Communication Systems — how information spreads.
5. Hockey Bible — league-specific hockey rules and rulebooks.
6. UI / UX Bible — how the player experiences the simulation.

For every milestone, update all applicable volumes before production implementation begins.
