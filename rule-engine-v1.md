# Hockey GM Legacy – Official Codex Structure

The Codex is organized into six major volumes:

1. Game Bible — what the player experiences.
2. Technical Bible — how the engine works.
3. AI Bible — how people think.
4. Communication Systems — how information spreads.
5. Hockey Bible — hockey-specific rules and rulebooks.
6. UI / UX Bible — how the player experiences the simulation.

## Core Development Rule

Every major feature must be reviewed across all applicable volumes before implementation begins.

## Core Philosophy

Hockey GM Legacy is not simply a hockey management game. It is a living hockey universe where believable people make understandable decisions, organizations evolve, history is remembered, and every career leaves a lasting legacy.
