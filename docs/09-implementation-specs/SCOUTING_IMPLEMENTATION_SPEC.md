# Milestone 004 – Scouting Implementation Spec

## Purpose

Scouting gathers information.

It does not reveal truth.

The goal of scouting is to reduce uncertainty, not eliminate it.

## v1 Goal

- Hire scouts.
- Assign scouts.
- Scout leagues.
- Scout tournaments.
- Scout individual players.
- Produce imperfect reports.
- Track confidence.
- Allow the GM to personally scout players.
- Build draft boards.
- Build organizational knowledge.

## Core Rule

The player never sees true player ratings by default.

## Scouting Output

Reports should prioritize language:

- Facts
- Observations
- Opinions
- Unknowns
- Confidence

## Player Dossier Sections

- Overview
- Facts
- Scouting Reports
- Analytics
- Medical
- Character
- Relationships
- Career
- History
- GM Notebook
