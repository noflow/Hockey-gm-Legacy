# Milestone 003 – Owners Implementation Spec

## Purpose

The Owner System defines the relationship between the GM and ownership.

Owners provide:

- Budgets
- Goals
- Expectations
- Pressure
- Accountability
- Career consequences

Owners do not approve every hockey decision.

## v1 Goal

- Generate owners.
- Assign owners to organizations.
- Set preseason goals.
- Set budgets.
- Track trust.
- Track confidence.
- Track patience.
- Track GM autonomy.
- Evaluate the GM.
- Extend, warn, or fire the GM.

## Owner Archetypes

- Builder
- Competitor
- Community Owner
- Investor
- Traditionalist
- Innovator

## Autonomy Levels

- Low
- Normal
- High
- Full Hockey Control

## Owner Events

- owner_generated
- owner_assigned
- budget_approved

- gm_warning
