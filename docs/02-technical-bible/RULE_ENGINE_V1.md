# Rule Engine v1

## Purpose

The Rule Engine allows every league to operate from configurable rulebooks instead of hardcoded hockey rules.

## Responsibilities

- Load rulebooks
- Validate roster moves
- Validate contracts
- Validate trades
- Validate player eligibility
- Determine draft order
- Determine playoff qualification
- Determine valid contract clauses
- Determine budget restrictions
- Determine player rights

## System Boundary

The Rule Engine answers whether something is legal.

The Human Intelligence System decides whether a person wants to do it.

The Budget System answers whether it is affordable.

The Event Engine records what happened.
