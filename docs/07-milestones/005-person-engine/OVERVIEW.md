# Milestone 005 – Person Engine

## Purpose

Build the shared human foundation for every person in Hockey GM Legacy.

The Person Engine supports players, owners, scouts, coaches, GMs, agents, parents, staff, media, and league executives.

Everyone is a Person first. Roles are attached to people.

## Version 1 Scope

- Person model
- Identity
- Gender
- Birth date
- Age calculation
- Nationality
- Birthplace
- Status
- Role system
- Multiple roles per person
- Role start/end dates
- Basic reputation
- Basic personality profile
- Career timeline entries

## Not Version 1

- Full relationship engine
- Full memory engine
- Family trees
- Death/aging simulation
- Deep personality evolution
- Full career AI
- Player development
- Contracts
- Recruiting

## Design Rule

A person should keep the same identity forever.

A player who becomes a scout is still the same Person.
