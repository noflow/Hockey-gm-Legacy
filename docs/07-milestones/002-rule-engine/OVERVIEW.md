# Milestone 002 – Rule Engine

## Purpose

Make every league operate from configurable rulebooks instead of hardcoded hockey rules.

## v1 Scope

- Junior Rulebook v1
- Roster validation
- Contract validation
- Draft validation
- Playoff validation
- Budget validation

## Not v1

- Full NHL CBA
- Arbitration
- Offer sheets
- LTIR
- Retained salary
- Waivers
- Expansion draft
