# Role Intelligence

Every role uses the same Human Intelligence System but values different things.

## Players

Value role, opportunity, development, winning, money, family, respect.

## Coaches

Value systems, winning, development, locker room, GM trust.

## Scouts

Value accuracy, confidence, preparation, relationships, viewings, character, medical risk.

## GMs

Value organization health, owner trust, development, draft, trades, finances, long-term success.

## Owners

Value their own definition of success: winning, money, legacy, community, trust, and pride.
