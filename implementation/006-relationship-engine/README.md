# Milestone 006 – Relationship Engine

## Objective

Create the relationship foundation for the Legacy Engine.

## Deliverables

- Directional Relationship model
- Relationship dimensions
- Relationship history
- Value changes
- Value clamping
- Last interaction tracking
- Basic decay
- Unit tests

## Acceptance

The Relationship Engine is complete when relationships can be created, changed, decayed, and tested independently from UI/Godot.
